## Fun job hunting!

I created an extension that allows you to add and keep track of jobs via a list. 
It's colorful and interactive. You can:
- Add or remove a job
- Right click on a job you find, and you will see a new menu option to add to job seeker.
  This is nice to use for jobs found randomly out in the wild. For example, a company website and not a job board.
- If you install ollama locally, you can upload the text of your resume. Ollam will then:
    - Read your resume and job descriptions and sort your list with a 1-5 star rating.
    - Rank the job descriptions that are most closely aligned with your resume. This will help you target positions that you will likely have better odds
      of landing an interview.

llama3.2 is just fine if you are wondering. 

Storage: Storage is currently local storage, If you uninstall the extenstion, you will lose your hard efforts.

## Installing

1. If you want sortability, install Ollama locally --> Install llama3.2
2. Copy this repo.
3. Go to google--> extensions -->manage extensions -->Load unpacked
4. Make sure the extension is toggles on, and it should work!
